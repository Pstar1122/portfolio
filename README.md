# EduHub - School Portal Landing Page

A modern, responsive landing page for an educational portal built with HTML, CSS, and JavaScript.

## Features

- 🎨 **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- ✨ **Modern UI** - Clean and professional design with smooth animations
- 📚 **Multiple Sections**:
  - Navigation bar with sticky positioning
  - Hero section with call-to-action
  - Features showcase
  - Popular courses grid
  - Contact form
  - Footer with social links
- 🎯 **Interactive Elements**:
  - Smooth scroll navigation
  - Form validation and submission
  - Scroll animations
  - Hover effects
  - Button interactions

## File Structure

```
├── index.html      # Main HTML structure
├── styles.css      # Complete styling with Tailwind, Bootstrap, and custom CSS
├── script.js       # Interactive JavaScript functionality
└── README.md       # This file
```

## Customization

### Change Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... more colors ... */
}
```

### Update Content
Edit `index.html` to:
- Change course listings
- Update features
- Add/remove sections
- Customize text and images

### Add Functionality
Extend `script.js` to:
- Connect to a backend API
- Add more form validation
- Implement user authentication
- Add dynamic content loading

## JavaScript Features

- **Smooth Scroll Navigation** - All anchor links scroll smoothly
- **Form Handling** - Contact form with validation
- **Scroll Animations** - Cards animate in when scrolled into view
- **Intersection Observer** - Efficient scroll animation using modern APIs
- **Event Listeners** - Interactive button behaviors
